## 交付内容

路径：`evidence_mapping_agent/`（cockpit 镜像：`agents/evidence_mapping/`）

模块 4「证据梳理 Agent」**v0.2**：

- 统一外壳：`metadata` + `payload.evidence_map` + `self_review`
- 默认 LLM 评分（`prompts.py`），规则引擎兜底
- 每条假设独立绑定与强度；Self 用平均强度等综合，不是四选一
- 领域无关 Prompt；不因缺 quotes 扣分
- Schema / Demo / 单测仍可用

## 如何验收

```bash
cd evidence_mapping_agent
pip install -r requirements.txt
python examples/run_demo.py
python -m pytest tests -q
```

有 Key 时终端应看到 `scoring mode: llm`；无 Key 则走 rules。

## 总控怎么接

```python
from evidence_mapping import EvidenceMappingAgent
resp = EvidenceMappingAgent().run_dict(input_slice)
task_context["evidence_map"] = resp["payload"]["evidence_map"]
```

环境变量：`DASHSCOPE_API_KEY`（或 `QWEN_API_KEY` / `LLM_API_KEY`），可选 `EVIDENCE_MAPPING_MODE=auto|llm|rules`。

改代码后请重启 cockpit 后端再测。
